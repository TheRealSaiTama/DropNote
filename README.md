# Dropnote

Dropnote is a minimal, monochrome, local-first notes app shell built with React, Vite, TypeScript, Tailwind CSS, shadcn/ui patterns, and PWA support.

## Stack

- React 19
- Vite 7
- TypeScript
- Tailwind CSS v4
- shadcn/ui-style component setup
- vite-plugin-pwa

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
